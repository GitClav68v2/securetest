# Spiritel

Modern lead-generation website for **Spiritel** — perimeter security solutions built for small and medium businesses.

## What we do
- Open, modular perimeter security (no proprietary lock-in)
- Choose your own cameras, sensors, access control hardware
- Select your preferred monitoring / security provider
- We coordinate professional installation through vetted low-voltage distributors & installers
- Easy, affordable entry to high-quality perimeter protection for SMBs

## Project Status
- MVP: Single-page static site (HTML/CSS/JS)
- Inspiration: https://www.startechgrp.com/
- Goal: Clean, trustworthy, high-tech feel → focused on generating quotes/leads
