# Integration One

Lead-generation website for **Integration One** — perimeter security solutions for small and medium businesses.

## What We Do
- Open, modular perimeter security (no proprietary lock-in)
- Customer chooses their own cameras, sensors, access control hardware
- Customer selects their preferred monitoring provider
- Integration One coordinates professional installation via vetted low-voltage installers

## Tech Stack
- Plain HTML/CSS/JS — no build step, no framework
- Deployed via Cloudflare Workers (auto-deploy from GitHub main branch)
- Form submissions via Web3Forms → info@integrationone.net team inbox

## Structure
- CLAUDE.md → AI session instructions
- docs/launch.md → prioritized task list
- docs/run.md → task execution prompts
- index.html → homepage
- privacy.html → privacy policy
- assets/ → css, js, images
